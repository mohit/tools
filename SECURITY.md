# Security Policy

## Supported Tools

All tools in this repository are community-supported and maintained on a best-effort basis.

## Reporting a Vulnerability

If you believe you have found a security issue:

- For sensitive reports, contact the maintainer via their GitHub profile and avoid public disclosure.
- For non-sensitive issues, open a GitHub issue with clear reproduction steps.

Please include the tool name, version (or commit), and any relevant logs or screenshots.
